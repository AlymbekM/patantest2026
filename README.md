# 📋 Patan Test - Тесты по патологической анатомии

Веб-приложение для прохождения тестов с Google OAuth и Firebase.

## 🚀 Быстрый старт на Replit

1. Откройте: https://replit.com
2. Нажмите: `+ Create` → `Import from GitHub`
3. Вставьте: `https://github.com/AlymbekM/patantest2026`
4. Нажмите: `Import` и `Run`

## ✨ Функциональность

- ✅ Google OAuth 2.0
- ✅ Тренировочный режим (по блокам)
- ✅ Режим экзамена (50 случайных)
- ✅ Результаты с ошибками
- ✅ Адаптивный дизайн
